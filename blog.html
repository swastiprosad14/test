Question-1: What are the different ways to select an element in the DOM?

There are some way to select an element in the DOM using JavaScript, you can use methods like getElementById , getElementsByClassName , getElementsByTagName , querySelector and querySelectorAll .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is innerHTML retrieves the complete HTML content within an element, including tags, while innerText returns only the visible text content of an element , and textContent retrieves all the text content within an element, including whitespace and line breaks, regardless of styling or visibility, essentially giving you the raw text without HTML interpretation

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Event Bubbling is the behavior where when an event occurs on an element, it not only triggers an event handler on that element but also propagates upwards through the DOM hierarchy, triggering event handlers on its parent elements until it reaches the root of the document

Question-5: How do you create, add, and remove elements using JavaScript?

I can create new element in JS using document.createElement() and then add it to the DOM with appendChild() or insertBefore(). To remove an element, use removeChild() or parentNode.removeChild() on its parent element.
